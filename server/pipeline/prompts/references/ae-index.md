# Índice das Aprendizagens Essenciais — 1.º Ciclo e Pré-escolar

Referência rápida para o Architect consultar ao gerar DocSpec-AM.
Ficheiros completos em: `vault/documentos-oficiais/aprendizagens-essenciais/`

## Como usar

O Architect deve:
1. Identificar ano e disciplina(s) relevantes para o tópico
2. Consultar os ficheiros AE correspondentes para extrair descritores específicos
3. Mapear para áreas de competência do Perfil do Aluno
4. Incluir referências no campo `curriculum` do DocSpec-AM
